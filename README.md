说明

- 本项目使用`gin`作为Web框架，`gorm`作为ORM框架，`sqlite`作为数据库，`jwt`作为鉴权方式
- PUT请求时传参`json`，请求头为`Content-Type: application/json`
- 密码使用`sha256`加密储存
- 支持跨域请求
- 接口采用RESTful风格
